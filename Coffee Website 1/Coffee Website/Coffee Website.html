ABOUT THIS PROJECT

            
 
*Code Overview*

Ay and Uz's Coffee Machine is a simple console application that allows users to order coffee, 
view previous orders, and exit. The code is organized into a single class called Program with 
several methods, each responsible for a specific task.

*Authentication Method*

The Authentication method is the first point of entry for the user. It checks if the user knows
the secret password (hardcoded for now). If they do, it unlocks the menu! This method is quite 
simple and could be improved by using a more secure approach, like hashing and salting passwords.

*Menu Method*

The Menu method shows the main menu options: order coffee, view previous orders, or exit. It's 
like a menu board at a real coffee shop! This method is straightforward and could be improved by
adding more options or submenus.

*OrderCoffee Method*

The OrderCoffee method takes the user's coffee order (type and size) and calculates the price. It's
like placing an order at the counter! This method is a bit longer, but it's still relatively simple.
It could be improved by adding more options, like flavorings or creamers.

*Size Method*

The Size method asks for the size preference (small, medium, or large) and adjusts the price. Like 
choosing a cup size! This method is similar to OrderCoffee and could be improved in the same ways.

*Restart Method*

The Restart method lets the user order again or exit the app. Like starting a new order or leaving 
the coffee shop! This method is simple and could be improved by adding a confirmation prompt before exiting.

*PreviousOrders Method*

The PreviousOrders method shows all previous orders made. Like looking at past receipts! This method
is straightforward and could be improved by adding more details, like order dates and totals.

*ReturnToMenu Method*

The ReturnToMenu method takes the user back to the main menu. Like returning to the menu board! This
method is simple and could be improved by adding a confirmation prompt before returning to the menu.

*Code Insights*

The code uses some simple techniques to get the job done, but there's room for improvement. For example:

*Security*

- The authentication method is super simple (just a hardcoded password). In a real app, we'd want
something more secure, like hashing and salting passwords.

*Menu and Order Processing*

- The menu and order processing logic could be improved. Right now, it's pretty straightforward, 
but we could add more features and flexibility. For example, we could add submenus for different coffee types or sizes.

*Code Reuse*

- There's some repeated code in the methods (like asking for user input). We could simplify that
by creating a separate method for getting user input.

*Error Handling*

- This application handles errors and input validation well but it can be improved. We'd want to add some checks and balances
to make sure the app doesn't crash or get confused. For example, we could add try-catch blocks to handle
exceptions and input validation better to ensure the user enters valid data.

*Modularity and Reusability*

- The code could be more modular and reusable. For example, we could create separate classes for orders, 
customers, and menu items. This would make the code more scalable and maintainable.

*Object-Oriented Programming*

- The code could use more object-oriented programming principles, like inheritance and
polymorphism. This would make the code more flexible and reusable.

Restart Method

This method allows the user to order again or go back to the menu.

                        public static void Restart()
                        {

                            // allowing the user to order again
                            while (true)
                            {

                                order = order + 1; //order number increases after every order
                                moneySum = moneySum + money;
                                builder.Append(orderSum + " at " + DateTime.Now + "\n"); //this line will add all the orders to a string so that it can be displayed
                                Console.WriteLine("Thank you! ,Do you want another order?(Y/N)");
                                string input = Console.ReadLine().ToLower();
                                
                                while (input != "y" && input != "n") 
                                {
                                    Console.WriteLine("Please enter valid option!");
                                    input = Console.ReadLine();
                                }
                                

                                if (input == "n") //if user enters no this code will execute and break the loop
                                {
                                    Console.Clear();
                                    Console.WriteLine("Thank you for ordering at Ay and Uz's Coffee Machine!");
                                    Console.WriteLine($"Order Details: {order} order(s)\n");
                                    Console.WriteLine(builder);//displays the orders
                                    Console.WriteLine($"Your final total will be R{Math.Round(moneySum, 3)}");
                                    ReturnToMenu();
                                    

                                }
                                
                                Console.Clear();
                                Console.WriteLine($"Welcome back {name}! ");
                                OrderCoffee();



                            }
                        } // this method allows you to restart the process
